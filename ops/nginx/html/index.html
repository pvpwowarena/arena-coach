Возможности

Всё, что нужно команде арены

Система собрана из трёх компонентов, которые работают вместе без лишних действий с вашей стороны.

⚡

Реал-тайм подсказки

Бот присылает советы по матчапу прямо в Discord DM в момент начала боя — опенер, post-trinket плей, ключевые CD.

📖

База знаний TBC

Матчапы 2v2/3v3 из топ-источников: Mirlol, tbcpvp.com, стримы. Каждый совет со ссылкой на оригинал.

🔒

Вайтлист-доступ

Только игроки из вашего списка. Доступ выдаётся через Discord-команду и привязывается к персонажу.

🛡

Не нарушает ToS

Только read-only телеметрия chat log. Никакого input injection, никакой автоматизации действий за игрока.

🤖

Discord-бот

Slash-команды: /matchup, /opener, /glossary — можно спросить совет вне боя тоже.

🪟

Простая установка

Аддон в папку WoW, bridge.exe рядом с bridge.env — и готово. Никаких драйверов и сложных настроек.
