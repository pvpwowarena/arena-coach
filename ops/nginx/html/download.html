WoW TBC Arena

⚔️ Arena Coach Bridge

Локальный демон для Windows. Читает WoW chat-log и отправляет события на сервер — ты получаешь подсказки прямо в Discord DM во время арены.

Скачать arena-bridge.exe

🚀 Установка за 4 шага

  1. Скачать arena-bridge.exe и положить в удобную папку (например C:\ArenaCoach\)
  2. Создать файл bridge.env в той же папке (токен получить у администратора в Discord)
  3. Запустить arena-bridge.exe --check-config для проверки
  4. Запустить arena-bridge.exe перед игрой — оставить окно открытым

📄 Пример bridge.env

# Скопируй в bridge.env рядом с arena-bridge.exe WOW_INSTALL_PATH=C:\Program Files (x86)\World of Warcraft\_classic_era_ BACKEND_URL=https://pvpwowarena.surprise4you.dev BRIDGE_BEARER_TOKEN=вставить-токен-от-администратора BRIDGE_PLAYER_NAME=ИмяПерсонажа

✅ Проверка

# В cmd или PowerShell из папки с .exe: arena-bridge.exe --check-config # Ожидаемый вывод: # ✓ Конфигурация корректна
⚠️ Токен выдаёт администратор командой /access add в Discord-сервере. Не передавай токен другим игрокам — он привязан к твоему персонажу.
