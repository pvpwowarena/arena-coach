Установка

Скачать Arena Coach

Два компонента: аддон в папку WoW и bridge.exe на Windows. Следуй инструкции — займёт 5 минут.

1

Аддон для WoW

WoW TBC 2.4.3
Скачать ArenaCoach.zip
GitHub Releases · посмотреть все версии
  • 1 Распаковать архив — внутри папка ArenaCoach
  • 2 Скопировать её в World of Warcraft\_classic_era_\Interface\AddOns\
  • 3 Запустить WoW (или /reload в игре) — аддон появится в списке аддонов
  • 4 Убедиться что аддон включён — маленький значок «connected» появится в правом углу экрана
2

Bridge — фоновая программа

Windows x64
Скачать arena-bridge.exe
GitHub Releases · посмотреть все версии
  • 1 Положить arena-bridge.exe в удобную папку, например C:\ArenaCoach\
  • 2 Создать файл bridge.env в той же папке — токен получить у администратора в Discord
  • 3 Проверить конфигурацию: запустить arena-bridge.exe --check-config
  • 4 Перед игрой запустить arena-bridge.exe и оставить окно открытым
Пример bridge.env
# Скопируй в bridge.env рядом с arena-bridge.exe # WOW_INSTALL_PATH = C:\Program Files (x86)\World of Warcraft\_classic_era_ BACKEND_URL = https://pvpwowarena.surprise4you.dev BRIDGE_BEARER_TOKEN = вставить-токен-от-администратора BRIDGE_PLAYER_NAME = ИмяПерсонажа
⚠ Токен выдаёт администратор командой /access add в Discord-сервере. Не передавай токен другим — он привязан к твоему персонажу и реалму.

✅ Проверка

Запусти в cmd или PowerShell из папки с .exe — должен увидеть зелёное подтверждение.

# В cmd / PowerShell: arena-bridge.exe --check-config # Ожидаемый вывод: ✓ WoW install path: found ✓ Backend reachable: 200 OK ✓ Token accepted ✓ Конфигурация корректна
